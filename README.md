# Wind Power Prediction using LSTM

## 📌 Overview
This project predicts wind turbine power output using SCADA data and LSTM (Long Short-Term Memory) neural networks.

A simplified SaDE-inspired optimization approach is used to improve model performance.

---

## 📊 Dataset
- Wind Turbine SCADA Dataset
- Features:
  - Wind Speed
  - Wind Direction
  - Theoretical Power
  - Actual Power

---

## ⚙️ Methodology
1. Data Cleaning & Preprocessing
2. Feature Scaling
3. Sequence Creation (Time-Series)
4. LSTM Model Training
5. Hyperparameter Optimization (SaDE-inspired)
6. Model Evaluation

---

## 📈 Results
- RMSE: ~420
- MAE: ~271
- R² Score: ~0.88

---

## 📉 Visualizations
- Wind Speed vs Power
- Correlation Heatmap
- Time Series Analysis
- Actual vs Predicted
- Residual Analysis

---

## 🔮 Prediction
The model can predict future wind power based on user input values.
